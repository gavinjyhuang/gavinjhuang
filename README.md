# gavinjhuang.com

Personal website for Gavin J. Huang - A simplified React version.

## Features

- Built with React 18
- React Router for navigation
- SCSS styling
- Responsive design
- About, Resume, Projects, Stats, and Contact pages

## Development

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Structure

- `/src/pages` - Main page components
- `/src/components` - Reusable components
- `/src/data` - Content data (resume, about, stats)
- `/src/layouts` - Layout components
- `/src/static` - SCSS styles
- `/public` - Static assets
